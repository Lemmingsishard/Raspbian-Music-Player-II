Music Not Stopping Upon Closing

If you're here you probably had a situation like this:
REEEEEEEEEEEEE why is this song still playing even through it should be closed. I HAAAAAATE having to open task manager every time I won't to close this damned program.

If you are experiencing this, you should make sure you either pause before closing, or use the Exit button in the File Menu instead of the X on the topright.
